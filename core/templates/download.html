Download for Windows

How to install on Windows

After downloading, Windows might display a SmartScreen alert.

Click "More info" and then "Run anyway".

Follow the installer as you normally would.

Download for macOS

How to install on macOS

Open the .dmg file.

Drag One Scriba.app into the Applications folder.

On first launch, macOS might block the app because the developer is not yet verified.

Go to System Settings → Privacy & Security.

Click "Open Anyway" next to the One Scriba alert.

Select "Open" to finish.
